### 实现功能
 1. 实现发布文章，评论，分类，上传图片，后台管理，登陆注册等功能。
 2. 基于MVVM开发，前端采用vue.js,页面使用UIkit，框架来自[廖雪峰](http://www.liaoxuefeng.com/wiki/001374738125095c955c1e6d8bb493182103fac9270762a000)，ORM使用sqlalchemy
